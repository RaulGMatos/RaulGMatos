Visual Basic Programming Windows Environment

This course gives students experience with an object-oriented, event-driven programming language using a modern integrated development environment. Students write programs utilizing basic data types, control structures, multiple forms, user-defined classes, arrays, and collections and access data from forms, files, and databases.
